# Ronin v4 (MacOS)

## Installation

```bash
# clone the repo
$ git clone https://github.com/plutoniumm/Sherlock.git

# go into app's directory:
$ cd myproject

# install dependencies
$ brew install smckit

$ pnpm i
# OR
$ npm i

# run
$ node PATH/server.js
$ npm run build

# Go to http://localhost:4000
```

Safari butler

``` 
xcrun safari-web-extension-converter ./Quartermaster/
```
