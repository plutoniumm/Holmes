## Installation

Follow these steps to install this project directory:

```bash
# clone the repo
$ git clone https://github.com/plutoniumm/Sherlock.git

# go into app's directory:
$ cd myproject

# install dependencies
$ brew install smckit

$ pnpm i
# OR
$ npm i

# run
$ node PATH/server.js
$ npm run build

# Go to http://localhost:4000

```
